## The &lt;binomen&gt; element

### Attributes

None

### Possible Subelements

- &lt;generic_name&gt;
- &lt;specific_name&gt;

### Examples

Below is shown an example of the `<binomen>` element.

```xml
<binomen>
    <generic_name></generic_name>
    <specific_name></specific_name>
</binomen>
```
